# netlify-site4

A React (Create React App) navigation dashboard with links to all bhenning.com / brianhenning.com sites and tools. Deployed to `site4.bhenning.com`.

## Tech Stack

- React 18 (Create React App)
- Netlify deployment

## Features

- Dashboard grid linking to all nine sites
- Dynamically resolves the base domain (`bhenning.com` or `brianhenning.com`)

## Setup

```bash
npm install
```

## Running

```bash
./run.sh
```

## Deployment

Deploys to Netlify via `netlify.toml`.
